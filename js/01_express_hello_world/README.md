# 03 – Express.js (JavaScript)

## 📄 Description
Basic Express server setup with one test route.

## 📦 Installation

```bash
npm init -y
npm install express
npm install --save-dev nodemon
```

## 📂 Structure

```
index.js
```

## 🧑‍💻 Scripts in package.json

```json
"scripts": {
  "dev": "nodemon index.js"
}
```

## ▶️ Run Server

```bash
npm run dev
```

## 📘 Concepts Covered

- Express basic server setup
- `app.get()` route
- Middleware: `express.json()`
